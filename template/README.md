# spotlight-templates-vite-vue3
spotlight scaffold vue3 template for typescript with vite<br>

## Use：

     spotlight vite vue3 my-project

     cd my-project

     npm install

     npm run dev

     npm run build

此模板是vue3+typescript环境<br>

1.配置在vite.config.ts 当中；<br>
2.内置Todo示例

Let's vue ，enjoy!
